本目录提供给大家FPGA实验环境的相关软件。

1. 在windows下的实验环境是齐全的，基本说明文档也有。
   在linux下的实验环境，也基本齐全，但缺少部分说明文档，比如tftp在linux上的搭建和使用没有。

2. 目录结构：
   +--flash_programmer/  : flash芯片编程软件
   |        
   |--gcc4.3/            : 交叉编译工具链下载与安装说明
   |
   |--tftp/              : tftp服务器搭建软件
   |
   |--uart_soft/         : 串口软件
   |
   |--vivado/            : Vivado下载链接与安装方法
   |
   |--modelsim/          : modelsim下载链接与安装方法
